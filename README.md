### 사용 언어

- Swift(SwiftIUI) - 상황과 필요에 따라 UIKit 사용

### 라이브러리 & 프레임워크

- Combine (비동기 이벤트 처리)
- Kingfisher (이미지)
- Alamofire (API 사용 - Router,Interseptor,request,response)
- FireStorage (이미지 데이터 저장 공간)
- MapKit,CoreLocation (지도사용 및 , 사용자 위치 표시)

### 아키텍쳐 패턴

- MVVM
